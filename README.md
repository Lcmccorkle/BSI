# BSI 📦 Bill of Lading → Inventory Extractor

A simple Streamlit web app that lets your team upload a Bill of Lading PDF and automatically extracts:

- BOL #
- Ship To (City, State)
- Product
- Quantity
- Packaging
- Lot Numbers

Then it generates a clean Excel row matching your inventory sheet format.
